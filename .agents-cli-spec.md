# Agent Spec — LifeShift Concierge

## Overview
LifeShift is a personal concierge agent for major life transitions. It helps users
plan career changes, relocations, new parenthood, and retirement with phased
plans, budget ranges, checklists, and milestone timelines.

## Capstone Track
**Concierge Agents** — Kaggle AI Agents: Intensive Vibe Coding Capstone Project

## Example Use Cases
- "I'm moving to Austin with two kids in 12 weeks — what should I do first?"
- "I'm switching careers from teaching to product management."
- "We're having our first baby in four months and need a plan."

## Tools Required
| Tool | Purpose |
|------|---------|
| `generate_transition_plan` | Phased plan by transition type |
| `estimate_budget` | Category budget ranges |
| `build_checklist` | Phase-specific task lists |
| `get_timeline_milestones` | Dated milestones from start date |

## Constraints & Safety Rules
- Redact SSN-like and card-like identifiers before model calls.
- Append reminders for financial/medical/legal topics.
- Plans are illustrative — not professional advice.

## Success Criteria
- User receives a structured plan with budget and next actions in one session.
- Multi-agent delegation routes to planner, budget, and checklist specialists.
- Eval cases pass with score >= 4 on response quality.

## ADK Features Demonstrated
1. Multi-agent orchestration (concierge + 3 specialists)
2. Custom tool use / function calling
3. **Agent skills** (SKILL.md playbooks + SkillToolset)
4. Security callbacks (PII redaction, advice disclaimer)
5. Agent evaluation dataset
